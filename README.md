# Pathfinding-Visualizer-with-Dijkstra-s-Algorithm
This is a Pathfinding Visualizer using Dijkstra's Algorithm, and commented is BFS  Controls:  First Left Click: Start Second Left Click: End Z: Undo Y: Redo Holding Left Click: Draw Wall Tiles/Traffic Tiles T: Traffic Mode C: Clear the visited nodes, &amp; keep the Walls/Traffic Tiles R: Resets the whole grid Spacebar: Finding the weighted path
